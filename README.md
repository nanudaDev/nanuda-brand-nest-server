## Description

- [픽쿡 / PICKCOOK] 위대한상사 픽쿡 백엔드 서버
- NEST.JS + TYPEORM + MYSQL 8.0+
- NAVER CLOUD 바탕으로 업로드 진행

## Running the app

````bash
# development
$ npm run start

# watch mode and development
$ npm run start:dev

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
````
